این ریپازیتوری محل آرشیو و سازمان‌دهی تمام مقاله‌هایی است که مطالعه می‌کنم و همهٔ ارائه‌هایی که در دانشگاه ارائه می‌دهم. هدف از ساخت این مخزن ایجاد یک ساختار منظم برای نگهداری فایل‌ها، یادداشت‌ها، خلاصه‌ها و متادیتای مرتبط با هر موضوع علمی است. هر مقاله یا ارائه در یک پوشهٔ مستقل همراه با فایل‌های مربوطه ذخیره می‌شود تا مرور، پیگیری و استفادهٔ دوباره از آنها ساده و سریع باشد.
